Class: Character

Character()

Represents the main playable character in the game. Extends the `MoveableObject` class and includes animations, sounds, and movement logic.

animate() → {void}

Handles the character's animation and movement updates. Continuously checks for keyboard inputs and updates the character's position, movement direction, and animation frames accordingly. Runs two intervals: - One for movement at 60 FPS. - One for animation frame updates every 50ms.
